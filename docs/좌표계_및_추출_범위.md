가능하면 **한 눈에** 보고 싶은 정보만 남긴 요약입니다.

---

## 1. 단위

- **길이**: μm  
  - `E_field_grid_coords.npy` (m) → 코드에서 ×1e6 하여 μm 사용
- **전기장 \(E\)**: **V/m**  
  - `E_field_4cycle.npy` 값 = 이미 V/m  
  - 시뮬레이션/시각화 = V/m 그대로 사용  
  - 컬러바 표시만 \(E \times 10^k\) 식으로 스케일해서 보여줄 수 있음
